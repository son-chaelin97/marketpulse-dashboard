# MarketPulse Dashboard
실시간 암호화폐 마켓 데이터를 기반으로 한 **코인 시세 대시보드**입니다.
Binance 24hr Ticker API를 사용해 주요 마켓 정보를 주기적으로 갱신하고,
가격 / 변동률 / 거래대금 기준으로 **정렬 가능한 마켓 리스트 UI**를 제공합니다.

---

## 🔍 주요 기능

### 실시간 마켓 데이터 조회
- Binance 24hr Ticker API 연동
- TanStack Query(`useQuery`)를 사용한 5초 주기 polling
- 로딩 / 에러 / 빈 상태 UI 분리 처리

### 마켓 리스트 UI
- 테이블 기반 마켓 리스트 구성
- 컬럼별 정렬 기능
    - 가격
    - 24시간 변동률
    - 거래대금
- 정렬 방향 3단계 토글 (내림차순 → 오름차순 → 초기 상태)
- 모바일 환경에서 거래대금 컬럼 숨김 처리

### 데이터 레이어 & UI 책임 분리
- `useMarkets` 훅:
    - 데이터 fetching, polling, query 상태 관리 전담
- UI 레이어:
    - 정렬 기준에 따라 **Top 50 마켓 선별 및 정렬**
    - 동일 데이터에 대한 표현 방식은 UI 상태로 관리

---

## 🛠 기술 스택
- Next.js

- Tailwind CSS

---
 
## 🧠 설계 포인트
- **Query Layer와 UI Layer의 책임 분리**
    - 서버에서 받아온 데이터의 “정체성”은 Query에서 관리
    - 정렬 기준, Top 50개 선택 등 “표현 방식”은 UI 상태로 분리
- 정렬 기준 변경 시 재요청 없이 클라이언트에서 처리
- 테이블 시맨틱을 활용해 접근성과 정렬 UX를 동시에 고려
